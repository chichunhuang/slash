Notion 資料匯入匯出

這主題說介紹 Notion 資料匯入的兩種概念，
一個是 Import 另一個是 embed。
Imoprt: 將資料建立副本或是截圖，所以內容是 detached。通常是將電子文件讀入 Notion 軟體中。
Embed: 將資料或第三方軟體畫面嵌入 Notion，與來源資料同步。可想成是

除了單純檔案匯入外，Notion 也與多種 Web 介面軟體整合。
個人比較常用的是 GWS 相關所以僅記錄與 Google 相關第三方軟體整合。

所以實際使用上可先決定要如何呈現、資料來源以及使用目的，再選用資料匯入的放式。

  • Notion 資料匯入官方參考

Notion 資料匯入種類  ​

匯入種類摘要

種類說明
Import(匯入)
File-Based		Notion 支援從多種來源匯入 常見格式 資料，包括 Markdown / HTML / CSV / Word / Excvel / CSV / PDF / Google Doc 檔案，以及 Evernote、Trello、Asana 等其他工具的內容
Embed
3rd-party		不支援 iframe 標籤，但允許 Embed 以區塊嵌入第三方軟體內容，例如 YouTube、Figma、Google Maps、Jira、Github 等等
Zip File批次將 File-Based 檔案壓縮後上傳
3rd-party軟體同步
Google Drive 同步		透過 Embed ，將 Google Docs、Sheets、Slides 等檔案嵌入 Notion 頁面中。並提供 即時預覽

常見檔案匯入(Imports)  ​

  • 單一檔案: 左側  Sidebar   🢂   Settings   🢂   Import   🢂   種類選取  
  • 批次匯入(zip): 左側  Sidebar   🢂   Settings   🢂   Import   🢂 最下方   Convert Zip to Pages   🢂   拖拉 zip 檔並上傳  
  • 註: 單一檔限制 25M 以內。
  • 常見類型文件:
    • word
    • excel
    • html
    • markdown: 僅限不含 databasse 的 Notion Page 可以 md format 匯出
    • csv: 單頁的 Notion Database 可以以 CSV 匯出。
    • PDF: 匯入文字說明與圖片。轉換成為已上傳檔為名的 Notion Page。

檔案嵌入(Embeds)  ​

  • Notion 嵌入(Embeds) 指的類似以 Iframe 嵌入，為即時資訊且受來源方相關限制。若授權被取消則出現 'Failed to Load'
  • 與檔案匯入另存資料副本不同。
  • 接受多種格式: HTML files / PDF files / Audio / Video / Images / 相關可接受的來源方清單
  • 建立方式
    • 線上電子檔 : 直接貼上   URL   🢂   選取 Embed  
    • 咒語(Prompt) :   /embed   🢂   貼入指定來源   🢂 填入  Embed Link   (若選擇 Upload 則變成匯入)
  • 其他操作
    •   拖拉  下方直角符號可調整可視範圍
    • 嵌入區塊右上   …   可 View Ooriginal 或 Replace Embedded Content

Google Drive 相關匯入/同步功能​

目前試用下，
功能會隨檔案種類以及呈現方式而在介面上有些許差異，
不知是否因為免費版，
所以部分功能發生無法 import 僅能同步預覽。
部分指令功能受限制: 出現 系統已封鎖這個應用程式，不然就回應 403。
另外，左側欄的 Settings 下 menu 中也出現兩個 Connections 。

  • 帳號連結:   Settings   🢂   Import   🢂   Google Docs   🢂   Connect Google Account   🢂   選擇帳號

Google Account 連結設定

* 插入 Google Drive 方式 (會要求連結 google 帳號) * 複製 Google doc link，直接貼在 Notion 🢂 可再細選[呈現種類](#GDrive_Display) * Block 填入   /Google Drive   指令，會開啟已連結帳號檔清單。 __Googel 線上文件網址__

Googel Drive 咒語


  • 插入 Google 文件呈現種類  
    • URL: 單純的 URL link 指向 google doc.
    • Preview: 文件首頁縮圖，點縮圖下方可進入 google 文件.
    • Mention: 與 URL link 類似，但 mouse over 時，會出現原文件的縮圖。與 Notion 的 Mentioned Page 不同.
    • Block 右側的 … 有提供呈現方式轉換。
    • Embedded: 將只份文件嵌入，但不可編輯，僅可瀏覽以及切換在文件頁面，同樣點預覽下方可進入轉導到原始檔案.

Notion 資料匯出​

資料匯出會因為 Browser/ Mobile App / Desktop / 匯出格式不同而略有差異
這邊主要紀錄的是 Browser 下的使用方式
在共編與團隊合作下，商業版 owner 可以關閉資料匯出功能。

匯出方式

  • 頁面右上   …   🢂   Export   可叫出匯出功能，並自訂匯出格式。
    • 可選擇格式、內容、紙張大小。
    • 以及 是否打包並匯出子頁面

特殊內容匯

  • HTML 格式可額外匯出 comments 與 mentions
  • Notion Database 僅能以 CSV format 格式匯出

Notion 資料遷移  ​

  • Notion 資料遷移至其他帳號  
  1. 以 Owner 登入工作區。
  2. 將您的所有頁面整合到一個頂級頁面下。
  3. 將 Workspace 完全存取權分享給指定 email/account。
  4. 以接收者帳號登入，並至  共享 (share)   頂層頁面。
  5. 由根頁面(或逐次選擇所有子頁面)，點擊   …   圖示 🢂   Move To   🢂   Private Pages  。

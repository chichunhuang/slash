# Python Boolean 

> Python 與 Javascript 一樣空值/零/空集合等部分情境也可直接視為布林使用。

簡單紀錄: 

|     | False                         |True                         |
|-----|-------------------------------|-----------------------------|
| 集合 | 空集合 |  非空集合        |
| 字串 | 空字串 |  非空字串        |
| 數值 |   零  | 非零數值          |
|  x  | None |   |

Notion Timeline View​

Timeline View 上可自行選擇要呈現的 Properties。
時間軸上的 點標記 可對應到指定 Data Property。
時間軸上也可使用 區間標記，需自行選取起時始間與結束時間。

  • Timeline View
    • 時間軸:   ...   🢂   Layout   🢂   Show Timeline by   指定時間軸上顯示的資訊主題。
    • 顯示區段調整: Timeline 右側的時間單位下拉快捷與 today 按鈕，用來管理 Timeline View 的橫向呈現 by week/month/quater/year。
    • 註: Calendar View 則是經由   ...   🢂   Layout   🢂   Show Calendar as Month/Week   。
    • 資料數: Timeline 可以自行設定版面上的資料豐度，隱藏 Row 數量或隱藏部分 Column 訊息。
      • Row:   ...   🢂   Load limit   可暫時限制 view 的縱向呈現，限制起始時一次可顯示的 rows 筆數。若需要可經由   Load More   展開。
      • Toggle: Timeline 左上的   >>   (show table)，可調出 toggled 資訊。
      • Column: 與 Database 其他 Views 相同，可選擇隱藏或顯示 Row 部分欄位:   ...   🢂   Layout   🢂   Properties   控制 column visibility。

Insect Totem

圖騰緣起:​

回想 1988~1989 年第一次接觸電腦。
還記得那是 Win31 轉 Windows 95 的年代。
也是 BBS 崛起的年代，那時候的我連傳水球都不會。
其實那時候...... 我連怎樣打注音都不會......
所以沒人願意接我的水球......因為我連 abcd 位置在那兒都找不到。

宿舍很多人都在玩踩地雷。這個不須鍵盤的遊戲--踩地雷也是我正式接觸電腦的開始。
學校計算機概論教的也是很紅很強的繪圖軟體的 小畫家 。
別懷疑就是他，大學生電腦課上的是小畫家。
好了，大家應該猜到了，我是一個老頭子。

那個年代，netscape 才剛主導整個瀏覽器世界，是當時的 Chrome/Firefox。
在那個 http 剛萌芽的遠古時代，大家都開始在做 烘培雞 。
需要一個 logo/icon，raster graphics 相關軟體對窮學生來說太貴了。
Vector graphics 才是王道，感謝 Corel Draw 教育版真的友善。
所以我開始學習 Corel Draw。買書上補習班......
我承認，我是先接觸盜版的。
不過畫 log 時我用的是 Corel Draw 9 教育版。
順便感謝學姊幫我自電腦展提回來，
我真的不知道 Corel Draw 9 會提供那麼多工具與教學文件。
辛苦妳了。 目前，我手邊目前還留著 Corel Draw 6/8/9 教育版的光碟作紀念。

OK, I majored in Entomology.
那時候宿舍很多人迷甲蟲，鍬形蟲為最。
烘培雞 要一個響亮的名字。
每周都要畫點圖。
學了 Corel Draw。
所以開始畫自己的 logo。
然後 昆蟲圖騰 / 昆蟲物語 出現了，當初是要用來當夠購網站的主頁使用。
網域也申請了，一切就緒。
所以網站就...... 關閉了!
當兵去囉~~~
退伍後，網域就被人買走了。 Fxxk!
